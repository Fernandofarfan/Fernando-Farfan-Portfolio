# Fernando Farfan - Portfolio Web 🌐 (v1.3.5)

Bienvenido/a a mi portafolio personal como **Desarrollador Full Stack** y **Data Analyst**. Este proyecto fue creado con el objetivo de mostrar mi experiencia, habilidades y proyectos desarrollados en diversas tecnologías web y de análisis de datos.

![Screenshot](./public/preview.png)

## 🚀 Tecnologías utilizadas

- **Frontend**: React, Tailwind CSS, Framer Motion, Sonner
- **Backend (Proyectos relacionados)**: Node.js, Python, Java + Spring Boot, .NET
- **Base de Datos**: MySQL, MongoDB, PostgreSQL, SQL Server
- **Herramientas y DevOps**: Git/GitHub, Docker, CI/CD (GitHub Actions), Vite, gh-pages

## 📁 Estructura del Proyecto

```bash
fernando-portfolio/
├── public/               # Archivos estáticos (PDFs, Imágenes)
├── src/
│   ├── components/       # Componentes (Sidebar, MobileMenu, etc.)
│   ├── sections/         # Secciones: Experience, Education, Projects, etc.
│   ├── data.js           # Todo el contenido dinámico (Multi-idioma)
│   ├── App.jsx           # Lógica principal y ruteo interno
│   └── main.jsx
├── index.html
├── package.json
└── README.md
```
